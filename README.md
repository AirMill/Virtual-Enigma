# Virtual Enigma

A graphical simulation of the historical Enigma cipher machine.

Built for educational purposes and experimentation with classical encryption.

---

## ✨ Features

- Functional Enigma simulation
- GUI interface
- Rotor configuration
- Cipher and decipher mode
- Educational demonstration tool
